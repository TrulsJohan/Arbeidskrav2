# Second-Hand Market - Console Application

## Project Description
This is a complete console-based second-hand marketplace built in C# as part of **Arbeidskrav 2** (Tema 2 - Object Oriented Programming) at Gokstad Akademiet.

Users can:
- Register and log in
- Create, browse, search, update and delete item listings
- Purchase items (with immediate review prompt)
- View transaction history and profile with average rating
- Leave reviews (1-6 terningkast)

The application fully meets all functional and technical requirements, including clean OOP design, LINQ queries, enums, generic collections, exception handling, and Git version control.
